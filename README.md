# PEMA Consulting

Intro projet

## Getting Started

### Build

```
git clone https://github.com/WildCodeSchool/lille-0920-js-p3-pema.git
cd lille-0920-js-p3-pema
npm install
```

### To start the project :

```
npm start
```

Open `localhost:3000` and enjoy !
